# 📝 과제

## 1. UI Audit  
와이어프레임에 사용된 UI 요소를 체계적으로 분석하고 구조화

- 제공된 5개 화면 분석  
- UI 요소 목록 정리  
- 컴포넌트 단위로 그룹핑  

---

## 2. Design Token 정의  
UI Audit을 기준으로 디자인 시스템 foundation 구축

- Color  
- Typography (폰트 선정 이유 포함)  
- Spacing  
- Radius / Elevation (optional)  

---

## 3. Component 설계  
재사용 가능한 UI 컴포넌트 설계

- 주요 컴포넌트 정의  
- Variants & States  
  - Default  
  - Hover  
  - Disabled  
  - Error  

---

## 4. High-Fidelity UI 구현  
정의한 토큰과 컴포넌트를 활용하여 화면 구현
